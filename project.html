Description

ExecKit is a set of software units which are assigned to simplify, automate the execution and its analysis of different kinds of tasks and commands. Especially, you may use it for:

  • daily/nightly builds and tests of software products
  • checking the serviceability of your services
  • preparation of test reports
  • tracking status of a product
  • increasing visibility of a software project
  • automation of many other routine tasks which you perform from time to time
  • ...

Download

For the installation of the ExecKit download the archive with all ExecKit's modules.
Note: read the documentation to check the prerequisites for the ExecKit and to get installation instructions.

Get support

Self-Help Resources for this project

Before contacting the ExecKit project regarding a support issue, we encourage you to try using any self-help materials that the ExecKit project provides, such as documentation and known bug lists and support requests.

Bug tracker

If the answer is not found than the best way to get help is by creating a new item in the Bugs Tracker.

Alternatives

If you believe the project may have a different support resource that would be better for submitting the particular issue, please consider the following alternatives:

  • Mail to the Administrator
  • Mail to the Administrator using WEB form
  • Create new support request
  • Create new feature request

Our time response

ExecKit is developed by enthusiasts in free time, so we can not guarantee the immediate response. However, we always try to consider bug reports and support requests as soon as possible prioritizing the support activities.

For developers

Join this project:

To join this project, please contact the project administrators of this project, as shown on the project summary page.

Get the source code:

Source code for this project may be available as downloads or through the Subversion SCM repository used by the project.

Uploading patch:

If you have troubleshooted the problem, prepared the correction and believe that it should be included to the official ExecKit distributive, please create new patch review request and attach the Patch to the tracker.

Note: Patch files are simply Unified-Diff files showing the differences between your working copy and the base revision.

Beta testing:

You are very welcome to download beta versions and provide your feedback, reporting proposals and new issues in bug tracker system used by the ExecKit project.

If you requested a feature, we are happy to deliver it to you firstly for beta testing before including to the final release version.

Update project web pages:

If you are a web page developer interested in this project, please consider reaching out to the project admin (per the "Join this project" section, above) to offer your assistance.
